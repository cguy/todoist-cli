todoist-cli --help

Usage: todoist  COMMAND [arg...]

Command line client for Todoist.com, ease-to-use tasks manager

Commands:
  init         initialize the todoist program to be connected to your todoist.com account
  list         list some data (projects, tasks, etc.) from your Todoist account
  add          add a task
  task         process a task

Run 'todoist COMMAND --help' for more information on a command.
